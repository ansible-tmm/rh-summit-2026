Demo Cheat Sheet

Segment 1 — 2019 (Task-based Automation)
Type issue1 on ACME Alert Center → single capacity alert fires
Navigate to AAP → Templates → click Launch on "Increase Web Server Capacity"
Watch playbook run → type fix1 on ACME Alert Center → alert resolved by AAP
Segment 2 — 2023 (Event-Driven Automation)
Type issue2 on ACME Alert Center → 7 critical alerts cascade in
Switch to AAP → Rulebook Activations → type eda → 7 events fire, 7 auto-remediation jobs run; open Rule Audit to see each rule with its job template
Type fix2 on ACME Alert Center → all alerts resolved by EDA
AIOps Tools — MCP + AAP Integration
Type issue1 on any AIOps tool (Instana, ServiceNow, or Splunk)
→ An observable issue appears (alert / incident / error spike)
Click the issue → Detail view shows MCP calling AAP for diagnostics + remediation
Each tool shows a different use case: microservice cascade, DB failover, config drift
Now Assist Chat — AI Chatroom Demo
Open Now Assist Chat → type fix1 anywhere on the page
→ AI agent streams 8 messages: analyzing incident, connecting to AAP via MCP, launching "Restore Web Application" job template on nostromo, playbook output, and confirming resolution
